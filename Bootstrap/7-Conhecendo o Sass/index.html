Conhecendo o Sass

Nessa aula você dará os seus primeiros passos com a linguagem Sass!

Estilos Aninhados

Com o Sass podemos aplicar estilos a elementos aninhados de forma intuitiva.

Mixins e Regras Avançadas

Mixins permitem incluir estilos definidos previamente, evitando repetição de código. Eles ainda permitem usar argumentos, garantindo uma flexibilidade ainda maior para criar regras poderosas.

Importação de Arquivos e Módulos

Com o Sass podemos organizar e modularizar facilmente nossos estilos, garantindo a qualidade e manutenibilidade da base de código.
